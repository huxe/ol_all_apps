WHAT IS THE APP ABOUT??

You will be able to generate single payment link for multiple invoices. Its like a batch payment for invoices

HOW IT WORKS

Step 1.

Select multiple invoices from tree view

Step 2.

From server actions click on "Batch Payment Generate Link"

If the user select different customers error will be a popup

If the user select not posted invoices error message will be a pop-up

Step 3.

Payment link is generated you can send it to your customer :)

Step 4.

Now customer will open that link and make a payment.



Step 5.

After paying by customer wait for a minute and your all selected invoices will be paid.

Also payment from customer have attatched these invoices.
